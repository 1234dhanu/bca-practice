  3. If you want to edit an existing document, which is the right HTTP method to use?

    1. GET
    2. POST
    3. PUT
    4. DELETE
    Ans: The correct HTTP method to use when editing an existing document is PUT.
    • GET is used for retrieving data.
    • POST is used for creating new data.
    • PUT is used for updating or editing existing data.
    • DELETE is used for removing data.

  8. List any five organization elements. Ans: In HTML, organization elements are used to structure and organize the content of a web page.

    • <header>: Represents introductory content, typically containing headings, logos, navigation menus, and other elements at the top of a web page or section.
    • <nav>: Defines a section of navigation links, typically used for menus or navigation bars.
    • <main>: Specifies the main content of the document or a specific section within the document. It usually excludes navigation, header, and footer elements.
    • <section>: Represents a thematic grouping of content, typically with a heading, used to divide the content of a document into sections or chapter.
    • <article>: Represents a self-contained piece of content that could be independently distributed or reused, such as a blog post, newspaper article, forum post, or comment.
