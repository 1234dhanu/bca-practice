8. Write CSS code that using :nth-child() to color even items to red and odd items to green. 

    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
        <li>Item 4</li>
        <li>Item 5</li>
    </ul>

    ANS:
        /* Color even items red */
        li:nth-child(even) {
        color: red;
        }

        /* Color odd items green */
        li:nth-child(odd) {
        color: green;
        }
